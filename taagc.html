1. Executive Summary

Company: TRIPLE A AHAL GLOBAL CONCEPT (TAAGC)

Project: Private Party Election Monitoring System – Bauchi State Rollout

Objective: Build a private, internal election monitoring system for a political party to track agents, monitor polling units, collate results, and detect problems in real-time – entirely independent of INEC or government systems.

Technology Stack: Firebase (Database, Auth, Storage) + Vercel (Hosting, Serverless Functions) + React Native (Mobile App) + React (Admin Dashboard)

  • Interface with INEC systems or portals

  • Publish results publicly or to external audiences

3. Project Objectives

3.1 Primary Objectives

# Objective Description
1 Agent Tracking Track all 4,074 agents in real-time on an interactive map with GPS locations
2 Results Collation Collate internal vote counts from PU → Ward → LGA → State levels
3 Victory Dashboard Display real-time victory probability across all levels
4 Anomaly Detection Automatically flag suspicious results (turnout variance, duplicates, missing evidence)
5 Incident Management Receive, track, and resolve incidents reported by agents
6 Coverage Optimization Identify uncovered PUs and enable rapid reassignment
7 Broadcast Communication Send instant messages to agents via SMS and in-app notifications
8 Device Approval Ensure only authorized computers access the dashboard
9 Audit Trail Maintain complete logs of all admin actions
