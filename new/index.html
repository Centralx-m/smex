System Features

Complete election monitoring solution for your party

📍

Real-Time Agent Tracking

Track all 4,074 agents on an interactive map with live GPS locations, check-in/out status, and battery levels.

📊

Results Collation

Collate internal vote counts from Polling Unit → Ward → LGA → State level in real-time with photo evidence.

🚨

Incident Management

Report, assign, escalate, and resolve incidents with photos, GPS location, and severity levels.

🔍

Anomaly Detection

Auto-flag suspicious results including turnout variance, duplicates, and missing evidence for investigation.

📱

Agent Mobile App

React Native app with OTP login, GPS tracking, offline support, photo capture, and push notifications.

🖥️

Admin Dashboard

Interactive map, victory dashboard, coverage heatmap, broadcast tool, device approval, and export reports.

🔒

Device Approval System

No unauthorized access – each computer must be approved by Super Admin with role-based access control.

🏛️

4-Computer Command Center

Dedicated stations for Lead Admin, Data Analyst, Operations Coordinator, and Backup & Incident Response.

📢

Broadcast Communication

Send instant messages to all agents, specific LGAs, or roles via SMS and in-app notifications.
